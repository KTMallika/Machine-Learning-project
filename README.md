Iris Flower Prediction System 🌸

About the Project
The Iris Flower Prediction System is a Machine Learning project developed using Python and Streamlit.  
This application predicts the species of an Iris flower based on flower measurements like sepal length, sepal width, petal length, and petal width.

The main goal of this project is to understand how Machine Learning algorithms work in real-world prediction systems and to create a simple, interactive, and user-friendly application.

Features
- Interactive web application using Streamlit
- Predicts Iris flower species instantly
- Automatic best K-value selection
- Clean and simple user interface
- Power BI dashboard included for visualization
- Beginner-friendly ML project

Technologies Used
- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Power BI

Machine Learning Algorithm:
K-Nearest Neighbors (KNN)
This project uses the K-Nearest Neighbors (KNN) algorithm for classification.

KNN works by comparing the input values with the nearest available data points and predicting the flower type based on similarity.  
It is one of the simplest and most effective supervised machine learning algorithms for classification problems.

Why KNN?
- Easy to understand and implement
- Works well for small datasets
- Provides good accuracy for classification tasks

How to Run the Project

Install Required Libraries
```bash
pip install -r requirements.txt
```

Run the Application
```bash
python -m streamlit run app.py
```

After running the command, the application will open in your browser.
